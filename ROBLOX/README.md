# Flipp Studios — Roblox License Loader

## Overview

Automatically generated from the Flipp Studios marketplace codebase.

- **Marketplace**: https://flippstudios.com
- **API Endpoint**: `/api/license/verify`
- **Support**: support@flippstudios.com

## Installation

### 1. Import into Roblox Studio

Insert the `Loader.server.lua`, `LicenseVerifier.lua`, and `Configuration.lua` into
`ServerScriptService` as a folder named `FlippStudios`:

```
ServerScriptService
  └── FlippStudios (Folder)
      ├── Configuration (ModuleScript)
      ├── LicenseVerifier (ModuleScript)
      └── Loader (Script)  ← starts automatically
```

### 2. Configure

Open `Configuration.lua` and set:

| Field | Where to find it |
|---|---|
| `LicenseKey` | Your Flipp Studios dashboard → Licenses |
| `ProductId` | Product page URL or dashboard → Products (e.g. `"1"`) |
| `ApiUrl` | Already set to `https://flippstudios.com` — only change if self-hosted |

### 3. Run

Publish/run your game. The Loader prints verification results to the console.

## What it does

1. On game start, sends a verification request to the marketplace API
2. Sends your `LicenseKey`, `ProductId`, `PlaceId`, `UniverseId`, and `CreatorId`
3. The API checks license validity, expiry, product match, and universe binding
4. On success: enables your content and checks for version updates
5. On failure: prints the specific reason and blocks execution (anti-leak)

## Response types

| API Response | What it means |
|---|---|
| `LICENSE_NOT_FOUND` | Key doesn't exist — check your license key |
| `LICENSE_REVOKED` | License was revoked by the seller |
| `LICENSE_EXPIRED` | Subscription has expired |
| `PRODUCT_MISMATCH` | License was issued for a different product |
| `UNIVERSE_MISMATCH` | License is bound to another universe — possible leak |
| `RATE_LIMIT_EXCEEDED` | Too many requests — wait and retry |

## For developers

After successful verification, the loader sets `_G.FlippStudios`:

```lua
-- Check in your game scripts:
if _G.FlippStudios and _G.FlippStudios.Verified then
    print("License OK for " .. _G.FlippStudios.ProductName)
    print("Latest version: " .. _G.FlippStudios.LatestVersion)
end
```

## Automatic configuration

These files were generated by scanning the existing marketplace project:

- **Domain**: `flippstudios.com` (from marketplace configuration)
- **API route**: `/api/license/verify` (pre-existing with rate limiting, anti-leak, version check)
- **Collections used**: `licenses`, `products`, `productVersions`, `orders`, `notifications`
- **License field**: `key`, `productId`, `status`, `universeId`, `creatorId`, `expiresAt`
